Active Challenge Engine

Every academy challenge asks first, then explains. Reading alone never awards Java XP.

Complete Teaching Loop

Learn by predicting, tracing, fixing, filling blanks, and applying for loops, while loops, arrays, methods, sequential search, and binary search inside the game.

For Loop Floor

Predict output, trace i, fix off-by-one bugs, fill loop conditions, and apply loops to scanning the stock market.

Market Scanner Unlock

Master for loops to unlock a scanner that explains how View Market traverses stock arrays with i < prices.length.

Arrays Floor

Predict array access, trace the last valid index, fix invalid indexes, fill array length expressions, and apply parallel arrays to the stock market.

Index Vision Unlock

Master arrays to unlock buy/sell hints that explain valid index ranges and how parallel arrays keep ticker, price, ID, owned shares, and risk aligned.

While Loop Floor

Predict output, trace loop-control variables, fix infinite loops, fill digit expressions, and decode market signals using while loops.

Signal Decoder Unlock

Master while loops to unlock market signal hints that explain digit processing with signal % 10 and signal / 10.

Sequential Search Floor

Trace checked indexes, predict found indexes, handle not-found cases, fill return statements, and apply linear search to stock tickers.

Ticker Finder Unlock

Master sequential search to unlock a buy-screen tool that scans tickers from first to last and shows comparison counts.

Binary Search Floor

Learn the sorted-data requirement, trace low/high/mid, update the search range, fill while-loop conditions, and apply binary search to stock IDs.

Fast Broker Unlock

Master binary search to unlock a buy-screen tool that searches sorted stock IDs using low, high, mid, and comparison counts.

Mixed Review Boss

Beat a 10-question boss fight that mixes loops, arrays, methods, sequential search, binary search, and stock-market applications.

Java Trading License

Pass the boss to unlock a 20% final-score bonus and prove mastery across the academy.

Methods Floor

Predict return values, trace parameters, fix method calls, fill method headers, and apply reusable methods to trading logic.

Trade Calculator Unlock

Master methods to unlock buy/sell previews that explain calculateTradeValue(index, shares), return values, and reusable game logic.

Mastery Unlocks

Correct answers build topic mastery and unlock trading abilities like Market Scanner, Signal Decoder, Index Vision, Trade Calculator, Ticker Finder, Fast Broker, and Java Trading License.

Developer Tests
