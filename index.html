Market Mayhem v0.2

Java Learning Edition is a terminal-based stock trading game that teaches loops, arrays, methods, sequential search, and binary search while you play.

MARKET MAYHEM
=============
Day: 1 / 15
Cash: $1000
Portfolio: $0
Net Worth: $1000
Java XP: 0
Learning Mode: ON

0. Exit
1. View Market
2. Advance Day
3. View Portfolio
4. Buy Stock
5. Sell Stock
6. Java Concept Map
7. Loops + Arrays Lab
8. Methods Lab
9. Search Lab
10. Toggle Learning Mode
99. Run Developer Tests

Run Locally

git clone https://github.com/BosssyPanda/MarketMayhem.git
cd MarketMayhem
javac MarketMayhem.java
java MarketMayhem

Learning Mode

Shows Java code snippets, loop traces, concept checks, and Java XP alongside the playable trading game.

Search Lab

Practise manual sequential search and binary search with comparison counts, sorted-data checks, and terminal traces.

Developer Tests

Runs beginner-friendly PASS/FAIL checks for index validation, portfolio math, and manually coded search helpers.
